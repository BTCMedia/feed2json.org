feed2json

This feed2json is a RSS/Atom feed to `JSONFeed` (jsonfeed.org) converter. It is open source and lives at appsattic/feed2json. It uses the feedparser library to do the initial parsing.

Various bits of coding could be better and a library should be extracted, but until a few people have used it, perhaps sent in a patch or pull request to make the conversion better, I'll leave it for the moment. This will happen when we have a battle-tested library. Also, when I figure out charsets and using iconv.

Conversion

Note: this installation is currently usable by the public. Depending on how it goes, I may end up either making it private and you can install your own version, or moving it to different hosting.
